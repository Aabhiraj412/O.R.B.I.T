## Getting Started
To get started with the project, follow the instructions below:

1. Clone the repository:
    ```sh
    git clone git@github.com:Aabhiraj412/O.R.B.I.T.git
    ```

2. Navigate to the project directory:
    ```sh
    cd O.R.B.I.T
    ```

3. Install dependencies for the backend:
    ```sh
    cd Backend
    
    #for npm users
    npm install
    
    #for bun users
    bun install
    ```

4. Install dependencies for the frontend:
    ```sh
    cd ../Frontend

    #for npm users
    npm install
    
    #for bun users
    bun install
    ```

4. Install dependencies for the app:
    ```sh
    cd ../App
    
    #for npm users
    npm install
    
    #for bun users
    bun install
    ```

5. Start the backend server:
    ```sh
    cd ../Backend
    
    #for npm users
    npm run dev

    #for bun users
    bun run dev
    ```

6. Start the frontend server:
    ```sh
    cd ../Frontend

    #for npm users
    npm run dev:react #ract frontend
    npm run dev:electron #elextron desktop app

    #for bun users
    bun run dev:react #ract frontend
    bun run dev:electron #elextron desktop app
    ```
7. Start the frontend server:
    ```sh
    cd ../App

    #for npn users
    npm run dev #for stable network
    npm run dev:unstable #for unstable network

    #for bun users
    bun run dev #for stable network
    bun run dev:unstable #for unstable network
   ```
